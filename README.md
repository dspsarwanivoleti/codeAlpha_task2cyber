# PhishShield - Phishing Awareness Training System

## Overview

PhishShield is an interactive cybersecurity awareness platform designed to educate users about phishing attacks, fake websites, and social engineering techniques. The system helps users recognize cyber threats, understand attacker tactics, and learn best practices to protect themselves from online scams.

The project combines educational content, real-world phishing examples, and interactive quizzes to improve cybersecurity awareness.

---

## Objectives

* Educate users about phishing attacks.
* Teach users how to identify phishing emails.
* Explain how fake websites deceive victims.
* Raise awareness about social engineering attacks.
* Provide practical cybersecurity tips.
* Assess user understanding through quizzes.

## Modules

### Module 1: Introduction to Phishing

Topics Covered:

* What is Phishing?
* Types of Phishing Attacks
* Why Phishing is Dangerous
* Common Attack Scenarios

---

### Module 2: Recognizing Phishing Emails

Users learn to identify:

* Suspicious Sender Addresses
* Urgent Language
* Unexpected Attachments
* Grammar and Spelling Errors
* Fake Login Requests
* Suspicious Links

Example:

Legitimate:

[support@company.com](mailto:support@company.com)

Phishing:

[support-company-security@gmail.com](mailto:support-company-security@gmail.com)

---

### Module 3: Fake Website Detection

Users learn to identify:

* Misspelled URLs
* Missing HTTPS
* Fake Login Pages
* Poor Website Design
* Unexpected Redirects

Example:

Legitimate:

https://www.amazon.com

Phishing:

http://amaz0n-security.xyz

---

### Module 4: Social Engineering Awareness

Topics Covered:

* Phishing
* Spear Phishing
* Smishing
* Vishing
* Baiting
* Pretexting
* Tailgating

Real-world attack scenarios are included for practical understanding.

---

### Module 5: Cybersecurity Best Practices

Recommendations:

* Enable Multi-Factor Authentication (MFA)
* Verify Sender Identity
* Check Website URLs Carefully
* Avoid Clicking Suspicious Links
* Keep Software Updated
* Use Strong Passwords
* Report Suspicious Emails

---

### Module 6: Interactive Quiz

The system evaluates user knowledge using multiple-choice questions.

Sample Question:

Which of the following is a common sign of a phishing email?

A. Urgent request for action

B. Official company domain

C. Verified contact information

D. Secure website certificate

Correct Answer:

A. Urgent request for action
